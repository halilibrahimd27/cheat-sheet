# Code of Conduct

This project follows the **Contributor Covenant** v2.1.

[Full text](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)

## TL;DR

- 🤝 **Be respectful** to people of all backgrounds and skill levels
- 💬 **Critique ideas, not people** — focus feedback on the work
- 🙏 **Give and receive feedback gracefully**
- 🚫 **Zero tolerance for**:
  - Harassment, insults, personal attacks
  - Discrimination on any basis
  - Sharing private info without consent
  - Promoting **malicious / illegal activities** with this project's content

## Reporting

If you witness a violation, contact the maintainer via DM on GitHub or
use GitHub's report feature.

## Enforcement

Maintainers may apply:
- Warning
- Temporary suspension from issues / PRs / discussions
- Permanent ban

## Scope

- All interactions in this repo (issues, PRs, discussions, commits)
- External communication representing this project
